Multiple search

If you have a list of companies and list of positions you want to search corresponding to that company in a csv file please select this search. For example : Companies of Interest = [google, microsoft, samsung] , position of interest - [CEO, Founder, HR]. If you have data like this choose this search. An example file can be found on the Multiple search page for more reference.
